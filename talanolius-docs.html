Talanolius GUI

A high-performance, memory-efficient GUI library for K2 language

Get Started View Demo

Why Choose Talanolius GUI?

Talanolius GUI combines ultra-fast performance with elegant design to create stunning desktop applications.

Ultra-Fast Rendering

Leverages K2's ramdisk caching for microsecond rendering times, even with complex UIs.

Memory Efficient

Uses advanced memory pooling to minimize allocations and reduce garbage collection pauses.

Beautiful by Default

Comes with a modern, customizable theme system that looks great out of the box.

Accessibility Built-in

Designed with accessibility in mind, including screen reader support and keyboard navigation.

Cross-Platform

Works seamlessly across Windows, macOS, Linux, and embedded systems with a single codebase.

Lightweight

Core library is just 120KB, with optional modules for advanced functionality.

simple_app.k2 
# Simple Talanolius GUI Application
import Talanolius

# Create a new application
app = Talanolius.Application.new("Hello Talanolius")

# Create a window
window = Talanolius.Window.new({
  title: "My First Talanolius App",
  width: 800,
  height: 600,
  resizable: true
})

# Create a layout
layout = Talanolius.VBox.new({
  spacing: 20,
  padding: 20
})

# Add a heading
heading = Talanolius.Label.new({
  text: "Welcome to Talanolius GUI",
  fontSize: 24,
  fontWeight: "bold"
})

# Add a text input
input = Talanolius.TextInput.new({
  placeholder: "Enter your name",
  width: 300
})

# Add a button
button = Talanolius.Button.new({
  text: "Say Hello",
  primary: true,
  onClick: function() {
    const name = input.getValue() || "World"
    Talanolius.Dialog.show({
      title: "Greeting",
      message: "Hello, " + name + "!",
      type: "info"
    })
  }
})

# Add components to layout
layout.add(heading)
layout.add(input)
layout.add(button)

# Set window content
window.setContent(layout)

# Show the window
window.show()

# Start the application
app.run()

Getting Started

Follow these simple steps to start building with Talanolius GUI

1

Install K2 Language

First, make sure you have K2 language installed on your system.

curl -sSL https://k2lang.org/install | bash
2

Install Talanolius GUI

Use the K2 package manager to install Talanolius GUI.

k2 install talanolius-gui
3

Create Your First App

Create a new file called hello_talanolius.k2 and add the following code:

import Talanolius app = Talanolius.Application.new("Hello") window = Talanolius.Window.new({ title: "Hello Talanolius" }) label = Talanolius.Label.new({ text: "Hello, World!" }) window.setContent(label) window.show() app.run()
4

Run Your App

Run your application using the K2 compiler.

k2 hello_talanolius.k2

UI Components

Talanolius GUI provides a rich set of UI components to build beautiful applications

Button

Interactive button with various styles and states.

Talanolius.Button.new({ text: "Click Me", primary: true, onClick: function() { ... } })

TextInput

Single-line text input field with validation support.

Talanolius.TextInput.new({ placeholder: "Enter text", onChange: function(value) { ... } })

Checkbox

Toggle control for boolean values.

Talanolius.Checkbox.new({ label: "Enable feature", checked: true, onChange: function(checked) { ... } })

Dropdown

Selection control for choosing from a list of options.

Talanolius.Dropdown.new({ options: ["Option 1", "Option 2", "Option 3"], onChange: function(selected) { ... } })

Slider

Control for selecting a value from a range.

Talanolius.Slider.new({ min: 0, max: 100, value: 50, onChange: function(value) { ... } })

DataGrid

Powerful grid for displaying and editing tabular data.

Talanolius.DataGrid.new({ columns: ["Name", "Age", "Email"], data: [["John", 30, "john@example.com"], ...] })
