Getting Started with K2 on Ubuntu

  1. Update Package Lists

    sudo apt update

  2. Install Required Dependencies

    sudo apt install build-essential libssl-dev

  3. Download K2

    wget https://k2lang.org/download/k2-latest.tar.gz

  4. Extract the Archive

    tar -xzf k2-latest.tar.gz
cd k2

  5. Install K2

    sudo ./install.sh

  6. Verify Installation

    k2 --version

  7. Create Your First K2 Program

    echo 'print("Hello from K2 on Ubuntu!")' > hello.k2
k2 hello.k2

How It Works

Our virtual machine technology allows you to run a complete Linux environment directly in your browser, giving you hands-on experience with K2 in a real-world setting.

WebAssembly Powered

Our VM runs on WebAssembly technology, providing near-native performance right in your browser without plugins or extensions.

Secure Sandbox

The VM runs in a secure sandbox environment, isolated from your computer's system, ensuring complete safety while you experiment.

Full Performance

Experience K2's ultra-fast execution times in a real Linux environment, with accurate performance metrics and timing.

Technical Details

  • OS: Ubuntu 22.04 LTS (Jammy Jellyfish) optimized for browser execution
  • Memory: 512MB RAM allocated to the virtual environment
  • Storage: 1GB virtual disk space for your files and installations
  • Persistence: Your changes are saved for the duration of your session
  • Pre-installed Tools: gcc, make, python, vim, nano, apt, and other Ubuntu development tools
