K3 for Windows includes the compiler, standard library, package manager, and Stellar UI framework.

Installation Instructions

Windows Installer

  1. Download the Windows installer (.exe) from above.
  2. Run the installer and follow the prompts.
  3. The installer will add K3 to your PATH automatically.
  4. Open Command Prompt or PowerShell and verify the installation:
k3 --version

Windows ZIP

  1. Download the Windows ZIP file from above.
  2. Extract the ZIP file to a location of your choice (e.g., C:\Program Files\K3).
  3. Add the bin directory to your PATH:
setx PATH "%PATH%;C:\Program Files\K3\bin"
  1. Open a new Command Prompt or PowerShell window and verify the installation:
k3 --version

Note: If you're using Windows 7 or earlier, you may need to install the Visual C++ Redistributable for Visual Studio 2019.

Installation Instructions

macOS Installer

  1. Download the macOS installer (.pkg) from above.
  2. Open the package and follow the installation wizard.
  3. The installer will add K3 to your PATH automatically.
  4. Open Terminal and verify the installation:
k3 --version

macOS Tarball

  1. Download the macOS tarball (.tar.gz) from above.
  2. Extract it to a location of your choice:
tar -xzf k3-1.0.0-macos.tar.gz
  1. Move to a directory in your PATH:
sudo mv k3-1.0.0 /usr/local/k3
sudo ln -s /usr/local/k3/bin/k3 /usr/local/bin/k3
  1. Verify the installation:
k3 --version

Homebrew

Alternatively, you can use Homebrew to install K3:

brew install k3

Installation Instructions

Ubuntu/Debian

Using the .deb package:

sudo dpkg -i k3-1.0.0-amd64.deb
sudo apt-get install -f

Or using the package repository:

curl -fsSL https://k3-lang.org/gpg | sudo apt-key add -
sudo add-apt-repository "deb [arch=amd64] https://k3-lang.org/apt stable main"
sudo apt-get update
sudo apt-get install k3

Fedora/RHEL

Using the .rpm package:

sudo rpm -i k3-1.0.0-1.x86_64.rpm

Or using the package repository:

sudo dnf install -y dnf-plugins-core
sudo dnf config-manager --add-repo https://k3-lang.org/rpm/k3.repo
sudo dnf install k3

Arch Linux

Using the AUR:

yay -S k3

Generic Linux

  1. Download the Linux tarball (.tar.gz) from above.
  2. Extract it:
tar -xzf k3-1.0.0-linux-x64.tar.gz
  1. Move to a directory in your PATH:
sudo mv k3-1.0.0 /usr/local/k3
sudo ln -s /usr/local/k3/bin/k3 /usr/local/bin/k3
  1. Verify the installation:

Build Instructions

Prerequisites

To build K3 from source, you'll need:

  • C++ compiler (GCC 9+, Clang 10+, or MSVC 2019+)
  • CMake 3.15+
  • Python 3.6+
  • Git
  • LLVM 12+ (optional, for JIT compilation)

Building from Source

  1. Clone the repository:
git clone https://github.com/k3-lang/k3.git
cd k3
  1. Create a build directory:
mkdir build
cd build
  1. Configure and build:
cmake ..
cmake --build . --config Release
  1. Install (optional):
sudo cmake --install .

Note: For more detailed build instructions and options, see the BUILDING.md file in the repository.
