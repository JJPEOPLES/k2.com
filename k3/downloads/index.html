Installation Instructions

Windows Installer

  1. Download the Windows installer (.exe) from above.
  2. Run the installer and follow the prompts.
  3. The installer will add K3 to your PATH automatically.
  4. Open Command Prompt or PowerShell and verify the installation:
k3 --version

Windows ZIP

  1. Download the Windows ZIP file from above.
  2. Extract the ZIP file to a location of your choice (e.g., C:\Program Files\K3).
  3. Add the bin directory to your PATH:
setx PATH "%PATH%;C:\Program Files\K3\bin"
  1. Open a new Command Prompt or PowerShell window and verify the installation:
k3 --version

Note: If you're using Windows 7 or earlier, you may need to install the Visual C++ Redistributable for Visual Studio 2019.

Installation Instructions

Linux Package

  1. Download the Linux package (.zip) from above.
  2. Extract the archive:
unzip k3-1.0.0-linux-x64.zip
  1. Build K3:
cd k3-1.0.0-linux-x64/bin
./build_k3.sh
  1. Install K3:
sudo ./install_k3_linux.sh
  1. Verify the installation:
k3 --version

Note: K3 requires a C++17 compatible compiler (GCC 7+ or Clang 5+) and glibc 2.17 or newer.

Build Instructions

Building from Source

  1. Download the source code from above or clone the repository:
git clone https://github.com/k3lang/k3.git
cd k3
  1. Build K3:
# On Windows
cd bin
build_k3.bat

# On Linux/macOS
cd bin
./build_k3.sh
  1. Install K3 (optional):
# On Windows (run as administrator)
install_k3_windows.bat

# On Linux/macOS
sudo ./install_k3_linux.sh

Note: Building K3 from source requires a C++17 compatible compiler (GCC 7+, Clang 5+, or MSVC 2019+).
