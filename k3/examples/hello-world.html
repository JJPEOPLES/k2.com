Hello World Examples in K3

Learn how to write your first K3 programs with these simple Hello World examples.

Basic Hello World

The simplest K3 program is a classic "Hello, World!" example. This demonstrates the basic structure of a K3 program and how to print text to the console.

Simple Hello World

This is the most basic Hello World program in K3. It defines a main function that prints "Hello, World!" to the console.

hello.k3 
// Hello World in K3
fn main() {
    println("Hello, World!")
}
Hello, World!

Note: The main function is the entry point of a K3 program. When you run a K3 program, the main function is automatically called.

Hello World with Command-Line Arguments

This example shows how to access command-line arguments in K3. It greets the user by name if a name is provided as an argument, or defaults to "World" if no argument is given.

hello_args.k3 
// Hello World with command-line arguments
fn main(args: []string) {
    if args.length > 1 {
        println("Hello, {}!", args[1])
    } else {
        println("Hello, World!")
    }
}
$ k3 run hello_args.k3
Hello, World!

$ k3 run hello_args.k3 Alice
Hello, Alice!

Tip: In K3, args[0] is the name of the program itself, and args[1] is the first actual argument provided by the user.

Interactive Hello World

These examples show how to make your Hello World program more interactive by accepting user input.

Hello World with User Input

This example asks the user for their name and then greets them. It demonstrates how to read input from the console in K3.

hello_input.k3 
// Hello World with user input
fn main() {
    println("What is your name?")
    
    let name = readLine()
    
    if name.trim() == "" {
        println("Hello, stranger!")
    } else {
        println("Hello, {}!", name.trim())
    }
}
What is your name?
Alice
Hello, Alice!

Note: The readLine() function reads a line of text from standard input. The trim() method removes whitespace from the beginning and end of the string.

Web Hello World

K3 can also be used to build web applications. These examples show how to create a simple web server that responds with "Hello, World!".

Hello World Web Server

This example creates a simple HTTP server that responds with "Hello, World!" to all requests. It demonstrates K3's built-in HTTP server capabilities.

hello_web.k3 
// Hello World web server
import Http from "k3/http"

fn main() {
    let server = Http.createServer((req, res) => {
        res.setHeader("Content-Type", "text/html")
        res.write("<h1>Hello, World!</h1>")
        res.end()
    })
    
    server.listen(8080, () => {
        println("Server running at http://localhost:8080")
    })
}
Server running at http://localhost:8080

Tip: When you run this program, you can open your web browser and navigate to http://localhost:8080 to see the "Hello, World!" message.

Hello World with Stellar UI

This example creates a simple interactive web application using K3's Stellar UI framework. It displays a greeting that updates as the user types their name.

hello_stellar.k3 
// Hello World with Stellar UI
import Stellar from "stellar/ui"

class Greeting extends Stellar.Component {
    state = {
        name: "World"
    }
    
    fn updateName(event) {
        this.setState({ name: event.target.value })
    }
    
    fn render() -> Stellar.Element {
        return <div>
            <h1>Hello, {this.state.name}!</h1>
            <input 
                type="text" 
                value={this.state.name}
                onChange={this.updateName}
                placeholder="Enter your name"
            />
        </div>
    }
}

fn main() {
    Stellar.render(<Greeting />, "#app")
}

Note: This example uses JSX-like syntax for defining UI components. The Stellar UI framework is built into K3 and provides a powerful way to build interactive web applications.
